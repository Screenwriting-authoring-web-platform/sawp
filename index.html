Screenwriting authoring web platform

Description

This php written platform allows team based scriptwriting according with the standards set by television and film industries for movie and TV production, including fiction and nonfiction (commercials, documentaries, etc.).

Target audiences: Small-to-middle sized studios and production teams, individual artists and students involved in media production.

Features

  • Format text
  • Tag words in categories
  • generate breakdown reports (lists, table, statistics)
  • Export html,pdf,docx,odt,rtf
  • Import trelby,celtx
  • Screenplay revisions (history)
  • Easy 5 minute install using installer
  • Comments
  • Teamroles (Director,Artist,Observer) Observers can only read screenplay and post comments
  • Responsive design, hide components for smaller screens
  • Hotkeys for fast use
  • Spellchecking (depends on browser)
  • Multiple users organized in teams
  • Localisation
  • Settingspage for administrator

Requirements

  • PHP 5.4+
  • MySQL
  • Webserver
