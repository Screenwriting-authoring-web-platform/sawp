Description

The current team was once called project, so if you read project in the code or comments you know that this means team.

Structure

Multiple users can combine into one team.
In one team the users work on multiple screenplays.
A member of a team is called collaborator.

rights and roles

there are global and team based roles.

global

admin: Is allowed to edit all teams and users, can change site settings, can activate, ban and delete users
user: Normal user to work in teams.
An admin has also all userrights.

team

director: Can edit team, invite and kick collaborators, change collaborator roles.
artist: Can create screenplays and write them. Can also delete commentthreads and edit tagtree.
observer: Can only view screenplays and add comments.
A director has all artist and observer rights, a artist has all observerrights.
