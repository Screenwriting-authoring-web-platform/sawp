The intended use of this webapplication is to make it easy to write screenplays for small studios, schools and universities. It schould have only a few requirements at the server and be easy to use and install. Furthermore it is important that this tool is free (like a free beer and free to use for whatever you want (see licence)) and its sourcecode is available.
